#PantherActive

This app makes class room quiz fun and easy.This is the web (instructors) part of the quiz app.
There is a separate but related android app for students.

The app uses these technologies:

* AngularJS 1.5.x
* Firebase 3.0.x
* AngularFire 2.0.x
* Bootstrap 3.3.x
