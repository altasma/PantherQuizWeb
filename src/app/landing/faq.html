Teacher's Manual

How PantherQuiz Works? Offline Capability and More...

Can student's use the Teacher's web App?

No. Students use a separate Andorid mobile app. Students can downlad the app from google paly store PantherQuiz Android App

Is there a limit on the number of classes and/or students I manage on PantherQuiz?

No. You can have as many of classes and students as you may need, and it is free.

How long will my classes and quizzes be saved?

Right now there is no time limit , it will be saved as long as you do not delete it manually.

Internet disconneted or disrupted in the middle of working on quizzes or questions, then what?

No problem PantherQuiz still works fine as if there was no internet disruption, it will re- sync to the server upon reconnection.

All works will be saved locally and be updated automatically.

Do I need to stop working on what I have started upon internet disruption?

No you don't have to stop, you can just continue to work on what you have started.

But do not refresh your browser.

Can I have both student and teacher account with the same ID(email).

Yes you can use one account for both apps, but you will be treated as if you have two accounts.

For example you will not have access to a class or quiz from student's app even though you created the class or quiz from this account. You need to add yourself in to the class as a student.

Add Edit and Delete Quizzes

Add a New Quiz

How Can I Add a New Quiz?

1. Click Class Rooms -> Class Room List on Navigation menu

2. Click on the Quizzes Button of the respective class

3. Fill out all the required fields and click the New Quiz button.

To add questions to this quiz check read 'Manage Questions' section.

What is 'Challenge' from the New Quiz form?

Challenge is any combination of alphanumeric characters you may enter for each new quiz, so that only students with that code can take the quiz.

This is used especially for a in-class quiz to make sure only students in the class can take the quiz.

Can I leave the challenge field blank if I want?

Yes, you can leave it blank and all registered students can take the quiz with a the code.

Can I change the challenge at anytime?

Yes. You can edit the challenge at any time.

What is the lock/unlock button?

When a new quiz is created it is locked by default, means students will not see it on their dashboard even if the start date is available.

So when ever you want it to make the quiz available for the students you need to unlock it.

Can students take an unlocked quiz even though the start date is in the future?

No. Students will not able to take a quiz unless it is both unlocked and in a valid start and end date of a quiz.

Edit a Quiz

Can I Edit a Quiz?

Yes, you can edit the fields of a quiz at any time, make sure you entered a valid date range when editing a start and end dates.

Will the questions be lost or affected after editing a quiz?

No, Questions of a quiz will not be affected after editing a quiz.

Can I lock a quiz any time after I made it available?

Yes, you can lock a quiz after you made it availbale so students will not be able to acces it. But if a student already attempted the quiz he/she will still see the attmepted quiz on their dashboard.

Delete a Quiz

Can I delete a quiz from a class?

Yes, you can delete a quiz permanently.

All questions under this quiz will be deleted permanently as well.

Can I undo a deleted quiz?

No, you can't undo a deleted quiz. Make sure you want to do so before you delete a quiz.

How about questions of a quiz?

Check the 'Manage Questions' sections for all 'Question' related questions.

What formats of questions are supported?

Multiple choice and True/False questions are supported.

How many questions are allowed per quiz?

As many questions as you want.

How many choices can I use for a questions?

Atleast 2 and atmost 6 choices are allowed per question.

Can I pre-set the correct answer and will it be auto scored when students take the quiz?

Yes, you have to preset the correct answer by clicking on the correct answer choice.

When students submit a quiz they will automatically see their result.

Do I need to hit save every time I added a new question or choice?

Not necessarily, PantherQuiz keeps insync with the database while you work on the questions?

Edit Question/Choice

Can I edit a question?

Yes, you can edit a question at anytime and it will be synchronized with all connected students' devices in real time while you are editing.

Can I edit a question/choice while students are taking a quiz?

Yes, you can edit a quiz, question or a choice while students are taking a quiz, and it will be updated to all connected device in real time.

Make sure you let the students know that the quiz has been updated.

What will happen if I updated a quiz/question/choice after a student submits a quiz?

There will no be any effect on the stuident's submitted quiz.

Can I delete a question/choice?

Yes, you can delete a question/choice at anytime permanently.

Can I undo a deleted question or choice?

No, all delete actions are permanent.

What will happen if I delete a question or a choice after a student submitted a quiz?

There is no effect on the student's side after the student submitted that quiz.

Grading and Report Download

How grading works?

Are quizzes automatically graded?

Yes, all quizzes submiited on PantherQuiz are automatically socred immediately after a student submits.

How are scores calcualted?

Scores are calcualted on percentage based, for example -

numberOfQuestions = 10

studentsCorrectAnswer = 8

studentsScore = (studentsCorrectAnswer / numberOfQuestions) x 100

= (8/10)X 100

studentsScore = 80%.

Can I download the grade report of a class?

Yes, you can dowload a grade report of a class in csv(table) format.

How can I dowanload grade report for a class?

Go to Class Rooms List tab under Class Rooms tab and click on the Grades button for the respective class.

Update a student's score

Can I update student's score?

Yes, you can update students score.

Will the student's attempted quiz be affected after editing his/her score?

No, Only the score will be updated everything else will be the same as before.
